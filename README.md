# Radius Labs

Hands-on labs for [Radius](https://radapp.io) - the open-source application platform for cloud-native apps.

## Labs

| Lab                                                           | Description                                                                                                                                                 |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [001 - Radius Lab Session](./001-lab-session/) *(deprecated)* | ~~Get started with Radius: install the CLI, initialize an environment, author and deploy your first app, add Dapr, and move to Azure.~~                     |

The quickest way to run a lab is with **GitHub Codespaces**:
